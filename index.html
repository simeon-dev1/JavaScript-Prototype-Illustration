How It Works: The constructor function has a .prototype property that points to an object. When you create instances with new Student(), each instance gets a hidden link (called [[Prototype]]) to that same prototype object. This is how all instances share the same methods without duplicating them.

Key Takeaways

  • The CONSTRUCTOR has a .prototype property (you can see it)
  • The PROTOTYPE holds methods you want ALL instances to share
  • Each INSTANCE has a hidden [[Prototype]] link to that shared object
  • All instances point to the SAME prototype object (memory efficient!)
  • Use Object.getPrototypeOf(instance) to see the link
